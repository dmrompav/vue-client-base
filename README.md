# VUE-CLIENT-BASE

## Description

Simple client app with [Vue](https://vuejs.org/) (login-logout, jwt).

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development mode
$ yarn run dev

# production mode
$ yarn run prod
```
