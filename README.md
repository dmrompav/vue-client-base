# VUE-CLIENT-BASE

## Description

Simple client app with [Vue](https://github.com/nestjs/nest) (login-logout, jwt).

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
